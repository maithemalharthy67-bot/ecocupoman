## How to Run
1. Open the `ecocup_project` folder in **VS Code**
2. Install the **Live Server** extension (if not already installed)
3. Right-click `index.html` → **Open with Live Server**
4. The website opens at `http://127.0.0.1:5500`

## Pages
- **Home** — Hero, features, impact calculator, process steps, vision & mission, SDGs, contact
- **Products** — Scrollable bottle cards (Individual & Organization modes)
- **Cart** — Add/remove items, promo codes
- **Checkout** — Delivery info + payment
- **Order Confirmation** — Success page
